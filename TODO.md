今回新しく、以下のようなアプリケーションを作成したいと考えています。

部屋のレイアウトを定義し、その上に大きさを設定した家具を自由に配置できるようにすることで、レイアウトを事前に確認できるようなアプリケーションです。

### 機能一覧
- 家具の登録
  - 全て家具は長方形とする
  - 縦の長さと横の長さを入力できるようにする
  - 家具に名前をつけられるようにする

- 家具の移動
  - 家具は自由に移動・回転・削除ができるようになっている。


### 自宅のレイアウトの登録
  - この部分のシステム要件が固まっていない。部屋の形は綺麗な長方形ではないことが多いので、GeofenceのPolygonのような形式で登録するか、他に良い方法があれば相談したい。

### アーキテクチャ
  - webアプリケーションとして動くものを想定している。
  - 家具の情報などは必要に応じてFirebaseに登録できるようにしたい。
  - まずはFirebase接続せず、ローカルで動くようなものでOK
  - Frontend / Next, Typescript
  - Backend (必要なら) Hono Node.js Typescript
  - Infura (必要なら) GCP, terraform
これでお願いします。
